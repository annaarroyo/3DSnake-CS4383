# 3DSnake-CS4383

1.) Download project folder and open Project/Solution with 3DSnakeGame.sln on Visual Studio
2.) Once loaded into VS, run project and press space bar to begin moving snake
3.) Use the "WASD" keys to move the snake around the grid
4.) Collect food objects randomly populated on the grid to increase snake length
5.) As the snake's length increases, your score will go up
6.) If the snake hits the edges of the grid plane, or turns in on itself, the game will end
7.) Once game has ended, a popup window will display score
